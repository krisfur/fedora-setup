# Fedora Setup
Quick checklist with commands when setting up after a clean installation of Fedora Linux.

![Screenshot of the setup.](screenshot.png?raw=true "KDE Plasma Fedora")

## General order:
- update dnf
- kitty terminal
- google chrome
- git
- github CLI
- neovim
- vs-code
- pip
- rust
- C++
- node-js
- grub theme
- LazyVim
- fastfetch
- discord
